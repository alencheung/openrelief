# Application Security QA Resources

This directory contains templates, configuration files, and reference materials
for the Application Security QA skill.

## Contents

### Configuration Templates

- `semgrep-rules/` - Custom Semgrep security rules
- `snyk-config/` - Snyk configuration templates
- `zap-config/` - OWASP ZAP configuration files
- `ci-templates/` - CI/CD pipeline templates for security integration

### Report Templates

- `sarif-templates/` - SARIF report templates
- `html-templates/` - HTML report templates
- `compliance-templates/` - Compliance report templates

### Reference Materials

- `owasp-mapping/` - OWASP Top 10 mapping references
- `cwe-reference/` - Common Weakness Enumeration references
- `nist-controls/` - NIST security control mappings
- `remediation-guides/` - Security vulnerability remediation guides

### Security Checklists

- `pre-commit-checklist.md` - Pre-commit security validation checklist
- `deployment-security.md` - Deployment security validation checklist
- `compliance-checklist.md` - Security compliance checklist

## Usage

Copy templates from this directory to your project and customize according to
your specific security requirements and compliance standards.
